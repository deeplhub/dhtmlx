### 主要案例的：
Scheduler（日程）：BgSchedulerController、SchedulerController；<br>
Chart（图表）：ChartController；<br>
Gantt（甘特图）：IndexController


### 注意事项：
日程显示的字段有：id,text,start_date,end_date,scheduler_type,warn_date.
这几个字段是固定的，必须和上面的一致，否则查询不出数据。

<br><br>

warn_info这个参数是一个临时参数，主要接收从页面提醒日期接收字符串
